## Reglas para este proyecto

- **No verificar en browser preview**: Esta aplicación requiere autenticación con Firebase para acceder a cualquier página del dashboard. El preview siempre redirige al login y nunca se podrá verificar visualmente. No usar `preview_start`, `preview_screenshot`, `preview_snapshot` ni ninguna herramienta de preview para este proyecto. Hacer push directamente tras confirmar que no hay errores de compilación con los logs del servidor.
